This is the first post on the blog.

To write a new post, run:

```bash
./scripts/new-post.sh "My post title"
```

That creates a properly named file in `_posts/` with the front matter filled in.

To write a draft that isn't published yet, add `--draft`:

```bash
./scripts/new-post.sh "Half-baked idea" --draft
```

Drafts live in `_drafts/` and are skipped on build. Move the file to `_posts/` with a `YYYY-MM-DD-` prefix when it's ready.
